Introduction

The Box Model is very important concept in CSS browser because it determines how elements are positioned with in the browser window.Every element gets treated as a box in CSS.This help you understand how to create attractive layouts with CSS. According to the box model concept, every element on a page is a rectangular box and may have width, height, padding, borders, and margins.
